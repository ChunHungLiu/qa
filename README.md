# Top level

 - `install-devstack-xen.sh`: All-in-one script to install devstack in a domU
   on XenServer/XCP or other Xapi toolstack server
 - `xenserver-quantum-devstack.sh`: All-in-one script to set up xenserver with
   quantum.
